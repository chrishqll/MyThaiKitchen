**My Thai Kitchen**
23/24 International Baccalaureate Computer Science SL.
A custom food ordering and delivery Android application for a restaurant, written using Android Studio.

**Features**
User sign-up, log-in authentication
Real-time food stock updating
Delivery time estimates (not working)
Order history
Sends analytics and user purchases to Admin
Clean and attractive UI

**Built With**
Java
Android Studio
Android SDK
Gradle
Firebase

**Installation**
1. Clone the repository: git clone https://github.com/chrishqll/mythaikitchen.git
2. Open the project in Android Studio
3. Sync Gradle files: Click "Sync Project with Gradle Files"
4. Run the app: Connect an emulator or Android device and click run
